Overview

This software allows you to use and create ordering questions in Moodle, the most popular Learning Management System in the world. An ordering question appears in the browser as a jumbled list of sentences. Student drag the sentences around into the correct order and then click the "Check" button.

This software runs on any version of Moodle 2.x. It does not require on any other plugins. However, it is required by the Reader activity module.

Installation

$ cd /your/path/to/moodle/question/type
$ git clone git://github.com/gbateson/moodle-qtype_ordering ordering
